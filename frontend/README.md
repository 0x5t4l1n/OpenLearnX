# OpenLearnX

OpenLearnX is an AI-powered learning platform with adaptive quizzes, coding practice, course tracking, and dashboard analytics.

## Install

```bash
npm i openlearnx
```

## Project

This package contains the OpenLearnX frontend (Next.js).

## Quick Start (development)

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
